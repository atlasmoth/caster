# Running the app

## Create .env file with the following values

```bash
EXPO_PUBLIC_BASE_URL=<PUBLIC_BASE_URL_OF_API>
EXPO_PUBLIC_AUTH_BASE_URL=<PUBLIC_BASE_URL_OF_ORY_KRATOS>

```

## Start Expo

```bash
#web
npx expo start --web

#android
npx expo start --android

#ios
npx expo start --ios

```
